# Yoipace

飲酒量を増やすためではなく、急に酔いすぎないように飲む間隔を落とすための「酔い方ペースメーカー」MVPです。

## 主な機能

- 年齢、性別、身長、体重、飲酒頻度、顔の赤くなりやすさ、今日の体調を保存
- じわ酔い、会食モード、二次会温存、明日守るの4モード
- お酒の種類、量、度数、想定一口数から純アルコール量を計算
- モードとプロフィールに応じて飲み切り目安、一口間隔、水通知回数を算出
- Expo Notifications と Haptics による一口・水分補給の通知
- 飲酒開始、一杯追加、一時停止、再開、飲酒終了
- 当日の合計純アルコール量、飲酒時間、水通知回数、一口通知の遵守率
- 日別履歴と翌日の体調メモを AsyncStorage に保存

## 注意

このアプリは医療機器ではありません。血中アルコール濃度を正確に予測するものではなく、飲酒運転可否の判断には使えません。飲酒量を増やす提案は行わず、ペースを落とす、水を挟む、飲みすぎを防ぐ方向で設計しています。
